> 工程结构描述


## src/mlearning

机器学习目录，里面为《机器学习实战》的相关代码。

## src/spider

为自己平常使用的爬虫代码

## src/tensorflow

是深度学习的代码目录，里面包含了各种tensorflow的尝试代码。

## src/nlp

自然语言处理目录，里面为《python自然语言处理》的相关代码
